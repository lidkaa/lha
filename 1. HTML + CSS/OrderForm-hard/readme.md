## Wymagana wiedza

- HTML, CSS, Flexbox, Grid

## Cele główne

- [ ] Stwórz widok formularza do składania zamówienia
- [ ] Formularz ma mieć pola typu:
- radio
- checkbox
- select
- text
- textarea
- number
- phone
- email
- date
- [ ] Formularz powinien wpasowywać się w stylistykę projektu
- [ ] Pamiętaj o walidacji pól formularza
